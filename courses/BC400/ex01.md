# Exercise 01 — Navigating the ABAP Workbench

> [!NOTE]
> Rewritten for A4H (ABAP 2023 Docker)

**Scenario**

In this first exercise you will familiarise yourself with the ABAP Workbench — the integrated development environment inside SAP. You will use the Object Navigator (SE80), locate an existing program, and explore the key tools available to ABAP developers.

Prerequisites:
- **System:** A4H client 001
- **User:** `TRAIN-01` with developer authorisation

## Task 1: Open the Object Navigator

1. Log on to A4H client 001 as `TRAIN-01`.
2. Call transaction `SE80`.
3. In the left-hand dropdown, select **Repository Browser**.
4. From the second dropdown choose **Package**.
5. Enter `ZTRAINING` and press **Enter**.
6. Expand the package tree — locate the sub-node **Programs**.
7. Note the list of programs already in the training package.

## Task 2: Explore an existing program

1. In the package tree, double-click the program `ZTRAINING_HELLO`.
2. The source code opens in the editor on the right. Read through the code.
3. Identify and note:
   - The `REPORT` statement at the top.
   - Any `WRITE` statements.
   - The program attributes (click **Goto → Program Attributes**).
4. Check the **Title** and **Type** of the program (type `1` = executable report).

## Task 3: Run the program from the editor

1. With `ZTRAINING_HELLO` still open, press **F8** (or click the **Execute** button).
2. A new SAP session opens and runs the program.
3. Confirm the output appears in the list output screen.
4. Press **F3** to return to the editor.

## Task 4: Use the Where-Used List

1. In the source code, place the cursor on the word `WRITE`.
2. Press **Shift+F12** (or choose **Edit → Where-Used List**).
3. SAP searches all objects that use the `WRITE` statement in the package.
4. Review the results list — note any other programs that appear.
5. Close the where-used results.

> [!TIP]
> The Where-Used List is one of the most powerful tools in ABAP development. Use it whenever you rename a variable, class, or function module to find all callers.
